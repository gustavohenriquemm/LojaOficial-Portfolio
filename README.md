# 🎁 Presentes Especiais - Projeto de Portfólio

## 📋 Sobre o Projeto

Este é um **projeto demonstrativo** de e-commerce desenvolvido para portfólio. O site apresenta uma interface moderna, funcional e totalmente responsiva para uma loja virtual de presentes especiais, incluindo perfumes, cosméticos e kits.

⚠️ **IMPORTANTE**: Esta é uma versão de demonstração. Nenhuma transação financeira real será processada.

---

## ✨ Características

### Interface e Design
- 🎨 Design moderno e elegante com identidade visual em tons de rosa (#ffbdbd)
- 📱 Totalmente responsivo (mobile, tablet e desktop)
- 🖼️ Layout limpo e organizado
- 💫 Animações suaves e transições

### Funcionalidades Implementadas
- 🛒 Sistema de carrinho de compras (localStorage)
- 🔍 Busca de produtos em tempo real
- 📦 24 produtos demonstrativos com categorias
- 🏷️ Filtros por categoria
- 💳 Simulação de checkout (demonstrativa)
- 📊 Cálculo automático de totais

### Tecnologias Utilizadas
- **HTML5** - Estrutura semântica
- **CSS3** - Estilização avançada com Flexbox, Grid, animações
- **JavaScript** (Vanilla) - Lógica e interatividade
- **Font: Poppins** - Tipografia moderna do Google Fonts

## 🚀 Como Visualizar

### Opção 1: Abrir Localmente
1. Clone ou baixe o repositório
2. Navegue até a pasta `LojaOficial-Portfolio`
3. Abra o arquivo `index.html` em qualquer navegador moderno

### Opção 2: Live Server (VS Code)
1. Instale a extensão **Live Server** no VS Code
2. Clique com o botão direito em `index.html`
3. Selecione **"Open with Live Server"**
